# Cloud Bus Ticket System 🚌☁️

A cloud-based Bus Ticket Management System built using **HTML, CSS, JavaScript, and Firebase Firestore** as part of the UneeQ Cloud Computing Internship.

---

## Features

* Book and manage bus tickets
* Store passenger information
* Manage travel routes and dates
* Update ticket status
* Delete ticket records
* Real-time cloud synchronization using Firebase Firestore
* Responsive and user-friendly interface

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Firebase Firestore

---

## Cloud Functionality

This project uses Firebase Firestore as a cloud database to store and manage ticket information in real time. Data remains available across devices and sessions.

---

## System Functions

* Create ticket bookings
* View ticket records
* Update ticket status
* Delete bookings
* Real-time cloud database updates

---

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Ensure Firebase configuration is correctly added.
4. Start creating and managing ticket records.
